# AT-200 이미지 폴더 안내

이 폴더에는 매뉴얼에서 사용되는 이미지 파일들이 저장됩니다.

## 필요한 이미지 목록:

1. **system_overview.png** - 시스템 구성도 (경사계, 모니터, 전원 연결 다이어그램)
2. **display_interface.png** - 디스플레이 화면 인터페이스
3. **install_step1_01.png** - 센서 설치 방법
4. **product_01.png** - AT-200 제품 전체 이미지

## 이미지 추가 방법:

1. 위의 파일명으로 이미지를 저장하세요.
2. 권장 해상도: 800x600 픽셀 이상
3. 파일 형식: PNG 또는 JPG

## 플레이스홀더:

현재 HTML 파일의 이미지는 플레이스홀더로 설정되어 있습니다.
실제 이미지를 이 폴더에 추가하면 자동으로 표시됩니다.

임시로 다음 URL을 사용할 수 있습니다:
- https://via.placeholder.com/800x600
